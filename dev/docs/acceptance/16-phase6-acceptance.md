# Phase 6 文本增强验收

运行 `cargo run --release -- dev projects/test-project`，选择 **Phase 6 文本**。

1. 第一行同时出现普通、樱花色粗体、放大和斜体文字；样式只影响各自区段。
2. `蟹 / 桜 / 物語` 的假名叠加在正文正上方，不占用独立行高；正文基线保持一致，
   相邻长注音通过水平占位避让而不重叠。
3. `<br>` 强制换行；第二行 ruby 不与第一行重叠。
4. 打字机逐字显示时，尚未出现的字保留排版位置但不可见，ruby 与对应正文同时出现。
5. `-notend` 在文字完成后自动执行 shake；动画完成前不跳入下一句。
6. `-next` 与 `-concat` 合并为“前半句与后半句合并。”，speaker 不丢失。
7. `setTextbox:hide` 在 0.7 秒等待期间持续隐藏，`show` 后恢复；`:` 只隐藏到下一句。
8. 输入框出现后键入中日文，Backspace 可删除；空输入不能确认。确认后变量插值到下一句。

通过标准：字号切换、Hide、菜单淡出和全局扩散阴影继续作用于每个富文本/ruby glyph，且一行只在内容改变时重建。
